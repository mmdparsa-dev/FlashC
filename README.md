## 📌 About

**FlashC** is a modern application designed with performance, simplicity, and a clean user experience in mind.

Built using **Kotlin** and **Jetpack Compose**, FlashC focuses on providing a fast, reliable, and beautiful experience with a modern Android architecture.

1. Download the latest release.
2. Install the APK on your Android device.
3. Open FlashC and enjoy.

## 🏗️ Development

Clone the repository:

```bash
git clone https://github.com/mmdparsa-dev/FlashC
```

Open the project in Android Studio and build:

```bash
./gradlew assembleDebug
```
